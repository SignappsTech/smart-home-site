# Profili strank — Signapps (pametni dom)

8 profilov potencialnih kupcev za pametni dom na osnovi Home Assistant.
Namenjeni so pisanju oglasov, vsebin, prodajnih pogovorov in določanju, kateri
paket komu ponuditi. Grajeni na 5-osebnem kupčevem panelu iz [`COPY.md`](../../COPY.md)
+ 3 dodatni segmenti.

> **Zlato pravilo za vse profile:** nikoli ne rečemo »brez posegov v inštalacije«.
> Resnica je adut: *brez razbijanja sten — le minimalni, čisti električni poseg,
> ki ga opravi usposobljen monter*. En ujeti pretiravanje uniči zaupanje vseh.

**Pokritost paketov:** vstopni (Osnovni · od 2.000 €) → celovit (Napredni · od 4.500 €)
→ po meri (Premium · od 15.000 €).

**Kako brati profil:** *Kdo je* (kdo) · *Kaj si želi* (cilji) · *Kaj ga pritegne*
(kljuke = naše prednosti) · *Kaj ga odvrne* (ugovori/strahovi) · *Kako ga nagovorimo*
(sporočila + kanali) · *Verjeten paket* + *Ključni stavek*.
